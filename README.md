# Shunting Yard Algorithm & Calculator

Реализация алгоритма сортировочной станции (Shunting Yard) для преобразования арифметических выражений из инфиксной записи в обратную польскую нотацию (RPN, Reverse Polish Notation, ПОЛИЗ) и вычисления результатов.

## Описание

Проект состоит из двух основных компонентов:

1. **`shunting_yard.py`** — преобразует корректные арифметические выражения, записанные в инфиксной нотации, в обратную польскую нотацию (RPN, Reverse Polish Notation).
2. **`calc.py`** — вычисляет результат арифметического выражения, записанного в обратной польской нотации. 

### Поддерживаемые операции

- **Арифметические операции**: `+`, `-`, `*`, `/`
- **Возведение в степень**: `^` (правоассоциативная операция)
- **Функции**: `sin()` (один аргумент)
- **Константы**: `pi`
- **Круглые скобки**: `(` `)` для изменения приоритета операций

## Запуск тестов

Для запуска всех тестов используйте команду:

```bash
make test
```

## Использование через CLI

Оба скрипта читают входные данные из стандартного потока ввода (stdin) и выводит результат в стандартный поток вывода (stdout).

### Преобразование в RPN (shunting_yard.py)

```bash
echo "3 + 4 * 2" | python3 shunting_yard.py
python3 shunting_yard.py <<< "sin(pi/2) * 2"
echo "(3 + 4) ^ 2" | python3 shunting_yard.py
```

### Вычисление результата из RPN (calc.py)

```bash
echo "3 4 2 * +" | python3 calc.py
```

## Использование в качестве калькулятора

Проект можно использовать как калькулятор: записывать выражения в привычной инфиксной записи и получать результат вычисления.

Преобразовать выражение в RPN и сразу вычислить результат:

```bash
echo "3 + 4 * 2" | python3 shunting_yard.py | python3 calc.py
```

### Примеры вычислений

```bash
echo "10 - 5 / 2" | python3 shunting_yard.py | python3 calc.py
echo "(3 + 4) * (2 - 5)" | python3 shunting_yard.py | python3 calc.py
echo "sin(pi / 2)" | python3 shunting_yard.py | python3 calc.py
echo "pi * 2" | python3 shunting_yard.py | python3 calc.py
echo "2 ^ 3 ^ 2" | python3 shunting_yard.py | python3 calc.py
echo "3 + 4 * 2 / (1 - 5) ^ 2 ^ 3" | python3 shunting_yard.py | python3 calc.py
```
