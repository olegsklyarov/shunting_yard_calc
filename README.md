# Shunting Yard Algorithm

Реализация алгоритма сортировочной станции (Shunting Yard) для преобразования арифметических выражений из инфиксной записи в обратную польскую нотацию (ПОЛИЗ).

## Описание

Программа преобразует корректные арифметические выражения, записанные в инфиксной нотации, в обратную польскую нотацию (RPN, Reverse Polish Notation). 

### Поддерживаемые операции

- **Арифметические операции**: `+`, `-`, `*`, `/`
- **Возведение в степень**: `^` (правоассоциативная операция)
- **Функции**: `sin()` (один аргумент)
- **Константы**: `pi`
- **Круглые скобки**: `(` `)` для изменения приоритета операций

### Приоритет операций

1. Функции (приоритет 4)
2. Возведение в степень `^` (приоритет 3)
3. Умножение и деление `*`, `/` (приоритет 2)
4. Сложение и вычитание `+`, `-` (приоритет 1)

## Использование через CLI

Программа читает входные данные из стандартного потока ввода (stdin) и выводит результат в стандартный поток вывода (stdout).

### Примеры использования

#### Через pipe

```bash
echo "3 + 4 * 2" | python3 shunting_yard.py
```

Вывод:
```
3 4 2 * +
```

#### Через heredoc

```bash
python3 shunting_yard.py <<< "sin(2 + 3) * pi"
```

Вывод:
```
2 3 + sin pi *
```

## Использование в качестве калькулятора

Проект можно использовать как калькулятор: записывать выражения в привычной инфиксной записи и получать результат вычисления.

### Способ 1: Через pipe (рекомендуется)

Преобразовать выражение в ПОЛИЗ и сразу вычислить результат:

```bash
echo "3 + 4 * 2" | python3 shunting_yard.py | python3 calc.py
```
